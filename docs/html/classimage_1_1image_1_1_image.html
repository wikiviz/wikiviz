def image.image.Image.reload (   self) 
Reload image from disk. This facilitates re-loading of
images from disk in case the image content changes.

.. versionadded:: 1.3.0

Usage::

    im = Image(source = '1.jpg')
    # -- do something --
    im.reload()
    # image will be re-loaded from disk
